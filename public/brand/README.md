# Your brand assets go here

Replace the two placeholder files in this folder with your own — **keep the exact
same file names** so you never have to touch any code:

| File | What it is | Recommended size |
|---|---|---|
| `logo.png` | Your brokerage logo (shown in the footer). A transparent PNG looks best on the taupe footer. | ~440 × 120 px (wide), transparent background |
| `agent.jpg` | Your headshot (shown in the footer agent block). | Square, at least 400 × 400 px |

That's it. Drop your files in with these names and the site picks them up.

> Don't rename the files or change paths — `src/config/brand.config.ts` points at
> `/brand/logo.png` and `/brand/agent.jpg` by default. If you *do* want different
> names, update the `logo` and `agent.photo` values in that config file to match.

The building renderings (the 15 gallery images and the hero/intro videos) are the
**same on every copy** of this site and live elsewhere — you don't touch those.
